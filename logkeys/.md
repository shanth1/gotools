# `logkeys` Package

The `logkeys` package provides a set of constants for logger options.

## Usage

```go
package main

import (
	"errors"
	"time"
	"github.com/shanth1/gotools/log"
	"github.com/shanth1/gotools/logkeys"
)

func main() {
	logger := log.New(	)

	logger.Info().
		Str(logkeys.Service, "test").
		Str(logkeys.Version, "v1.0.0").
		Msg("test log")
}
```
